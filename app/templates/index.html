App

This app packages one screenplay scene into a RenderPackage: a practical workflow for AI video generation. You export a single downloadable file in .zip format containing a structured production kit.

  • Upload a screenplay file in .fountain format. Fountain is a plain-text screenplay format used by writers and developers because it is easy to read, version, and convert from script tools.
  • If your script has multiple scenes, choose the scene number you want to package. (If it has one scene, the app auto-selects scene 1.)
  • Select your provider and build the RenderPackage for that scene.
  • Output format is a .zip file containing your RenderPackage, including shot planning files, provider-ready prompts, Keeper Sheet, and audio post-production docs (dialogue, voice, and SFX cue sheets).

Creators

RenderScript is built for writer-directors and small teams who want to move fast in AI filmmaking without losing control of the cut. Instead of treating each generation as a one-off prompt, it gives you a repeatable production flow from shots and takes to keepers and edit.

The picture-first approach is central: focus on visual continuity first, then handle ADR/VO, SFX, and mix in post. That principle is used in professional workflows broadly, but RenderScript makes it explicit inside the package so teams can apply it consistently with AI-generated footage.

The result is fewer random rerolls, clearer creative decisions, and a cleaner path from screenplay scene to rough cut.

Under the hood, the AI-assisted RenderScript compiler turns screenplay structure into a stable package contract, so creators and developers can work from the same source of truth across prompts, shots, keepers, and post.

Developers

RenderScript gives you a reliable packaging layer between screenplay text and AI video tooling. The goal is simple: make creative intent machine-usable without locking teams to one provider or internal pipeline.

For product and pipeline teams, this means a stable handoff format you can validate, version, and automate in CI while creators keep working in familiar script language.

CLI quick start

  • python3 -m pip install -e .
  • renderscript package path/to/script.fountain --scene 1 --provider universal -o ./out/package.zip
  • Provider options: universal, runway.gen4_image_refs
