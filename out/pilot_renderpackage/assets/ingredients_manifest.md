# Ingredients Manifest

## Required asset categories

- Characters
- Locations
- Style references
- Props

## Capture checklist

- Keep framing and lighting consistent with intended shot style.
- Capture neutral, clean references with minimal motion blur.
- Capture at least one fallback reference per critical entity.
- Verify reference files are named consistently before packaging.

## Naming rules

- `char_A_ref_01`
- `loc_01_ref_01`
- `style_01_ref_01`
