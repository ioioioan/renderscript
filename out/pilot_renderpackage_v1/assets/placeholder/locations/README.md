# Locations Placeholder

Use this folder to stage reference images before generation.

## Naming examples

- `char_A_ref_01`
- `loc_01_ref_01`
- `style_01_ref_01`
- `prop_01_ref_01`

## Capture checklist

- Capture clean, front-facing references where applicable.
- Keep lighting and color temperature consistent.
- Avoid heavy motion blur and extreme occlusion.
- Keep wardrobe and key props consistent across captures.

## Provider limits

- Exact reference limits depend on provider.
- Runway Gen-4 Image References supports up to 3 active references.
