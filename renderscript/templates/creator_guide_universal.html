RenderPackage Creator Guide (Start here)

A RenderPackage is a scene-level production kit for AI video generation.

Golden rule

Picture-first.

No burned-in subtitles, captions, logos, watermarks, or on-screen text.

If text appears in a generated shot, discard that take and generate again.

Audio comes later: ADR / VO, sound effects, and mix are added after the rough cut.

Generation workflow

Open your AI video tool → apply references → paste the shot prompt → generate → mark keeper

Choose your starting workflow

Path A — Prompts-only (quick test)

Use this when you don’t have reference images yet.

Generate a few shots to test pacing and visual intent.

Expect more drift at this stage — this is normal.

Path B — References (best consistency)

Use this when you can provide reference images.

References improve consistency more than complex prompts.

This package shows which references each shot expects.

Workflow map

Refs: Reference images

Consistency comes from references. Start minimal, then refine.

Minimum reference set

• Style (look reference)

• Location

• Characters (one per character in the scene)

• Props (only if story‑critical)

Where to save your images

Save reference images into:

• assets/refs/styles/

• assets/refs/locations/

• assets/refs/characters/

• assets/refs/props/

Use the exact filenames listed in assets/ingredients_manifest.md (example: style_01_ref_01.png).

Takes: Generate shots

This is your dailies stage: generate multiple takes per shot, then pick a keeper.

Non‑negotiable

No burned-in subtitles or on-screen text. Keep picture clean.

If your tool adds text, reroll the take.

Shot loop (repeat for each shot_###)

1) Open the shot prompt block

2) Apply refs (if using Path B)

3) Generate 3–6 takes

4) Review fast

5) Mark a keeper

If drift rises: re-apply refs → simplify prompt → split the shot.

Keepers: Select, rate, iterate

Keeper discipline keeps you moving: pick what cuts well, reroll only what fails.

Keeper Sheet

Open keepers/scoring_sheet.csv. For each shot: keeper 0/1, rate character/location/style (1–5), add a short note.

keepers/scoring_sheet.csv

Final note

This isn’t “one prompt makes a film.”

It’s a repeatable director workflow: references + shot intent + keepers → less rework.
